
# Указатели и референции. Динамична памет

Оператор **&**

 - Приема променлива от тип Т.

 - Връща указател от тип T*.
 
 
```c++
  int number = 10;
  int* ptr = &number;
``` 


Оператор<b> * </b> (дерефериране)
- Приема указaтел от тип Т*
- Връща променлива от тип Т


```c++
  int number = 10;
  int* ptr = &number;
  int result = *ptr; 
  cout << result; //10
```
